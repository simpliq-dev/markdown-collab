# archive/

This folder contains materials from the build / ideation phase of `codex-collab`.

None of this content is required to run the VS Code extension, and it is excluded from the packaged VSIX via `.vscodeignore`.

Contents (may change):
- `PLAN.md` and `PROMPT-HELPER.md`: vibe-coding workflow artifacts used to scaffold the project
- `templates/`: early rule templates (superseded by `rules/`)
- `imgs/`: extra/oversize screenshots not intended to ship inside the VSIX
- `docs/`: draft listing copy / notes
